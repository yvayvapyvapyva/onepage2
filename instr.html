Элементы управления

Маршруты
Кнопка «Маршруты» — Открывает список маршрутов, отсортированных по папкам. Выберите маршрут — приложение загрузит точки и включит навигацию. Приблизьтесь к первой точке, приложение само озвучит команду и поведет по маршруту к последующим точкам.
Спутник / Схема — Переключает вид карты Спутник/Схема.
2D
Наклон карты — Переключает вид карты между 2D (вид сверху) и 3D (наклон 60°).
Симуляция проезда — Запускает симуляцию проезда по пути текущей точки. Сначала воспроизводится голосовая команда, затем начинается анимация движения по маршруту. Повторное нажатие ставит на паузу / возобновляет симуляцию.
Привязка карты к текущей геопозиции — Включает режим следования: карта автоматически центрируется по вашему местоположению и поворачивается по направлению движения. Повторное нажатие отключает.
КМ/Ч0
Скорость (спидометр) — Показывает текущую скорость в км/ч. Нажмите — откроются настройки оповещений.

Настройки оповещений:

Детекция ударов — при резком толчке (например проезд ямы или лежачего полицейского на большой скорости) приложение озвучит команду: «Сильный удар!». Что бы ученик привыкал объезжать ямы или сбавлять скорость. Чувствительность регулируется ползунком (0.5g–3g). Включена по умолчанию на 1g. На iphone может потребоваться дополнительное разрешение для использования акселерометра.

Максимальная скорость — при превышении заданного порога скорости (в км/ч) приложение озвучит команду: «Превышена максимальная скорость!». Полезно для контроля разгона до максимальной скорости. По умолчанию 58 км/ч, с учетом погрешности спидометра это около 60 км/ч на приборной панели.

Для каждой опции можно изменить текст голосовой команды.

Озвучка комментариев — Появляется, если в маршруте есть комментарии. Нажмите — озвучка и отображение комментариев отключатся (динамик перечеркнётся), повторное нажатие — включатся.

Блок команды — Слева номер точки, по середине текст команды, справа расстояние до точки. Нажмите — приложение озвучит команду голосом.
Предыдущая точка — При нажатии карта перемещается к предыдущей точке в режиме предпросмотра.
Следующая точка — При нажатии карта перемещается к следующей точке в режиме предпросмотра.
Выбрать
Выбрать точку целью — Появляется между стрелками при предпросмотре. При нажатии просматриваемая точка становится текущей целью.

Карта

Точки маршрута отображаются цветными маркерами. Текущая активная точка подсвечена зелёным кольцом. Цвет маркера соответствует типу команды. Нажмите на любую точку на карте — откроется предпросмотр с командой и путём. Нажмите в любом месте карты — предпросмотр закроется.

Голосовые команды

При приближении к целевой точке (<25 м) и совпадении направления (+-45°) приложение автоматически проигрывает голосовую команду.

Комментарии

Точки с комментарием отмечены красным контуром. Комментарий отображается сверху над командой. Нажмите, на него чтобы скрыть.

При автоматическом озвучивании команды (при приближении к точке или в симуляции) комментарий воспроизводится сразу после команды женским голосом.

Советы

  • Разрешите геолокацию для корректной работы навигации.
  • Приложение работает только при включенном экране.
  • При ограничениях работы интернета может не работать, используйте vpn.
  • В некоторых местах в определенное время могут глушить GPS и приложение не будет работать корректно.
