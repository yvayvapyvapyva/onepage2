Элементы управления

Маршруты
Кнопка «Маршруты» — Открывает список маршрутов, отсортированных по папкам. Выберите маршрут и приблизьтесь к первой точке маршрута ближе 25 метров, приложение озвучит команду и далее поведет по следующим точкам маршрута.
Настройки — Вид карты, настройки оповещений, комментарии, инструкция.

Вид карты — Спутник или Схема.

Настройки оповещений:

Детекция ударов — при резком толчке (яма, лежачий полицейский) приложение озвучит выбранную фразу. Чувствительность 0.5g–3g, по умолчанию 1g. Текст фразы можно изменить.

Превышение скорости — при превышении порога (км/ч) озвучивается заданная фраза. По умолчанию 58 км/ч. Текст можно изменить.

Комментарии — включение/отключение отображения и озвучивания комментариев к точкам.

Инструкция — открывает данную инструкцию.

Привязка к геопозиции — Карта автоматически центрируется по вашему местоположению и поворачивается по направлению движения. Повторное нажатие отключает.
КМ/Ч0
Скорость (спидометр) — Показывает текущую скорость в км/ч.

Нижняя панель

На перекрестке направо 50м
Блок команды — иконка точки с номером, текст команды, расстояние. Нажатие на иконку озвучит команду. Нажмите на текст команды — откроется список точек маршрута.
Навигационная панель:
< — переход к предыдущей точке
▶ — симуляция проезда: воспроизводится команда, затем анимация движения по пути точки. Повторное нажатие останавливает симуляцию.
> — переход к следующей точке

Советы

  • Разрешите геолокацию для корректной работы навигации.
  • Приложение работает только при включенном экране.
  • При ограничениях интернета может не работать — используйте VPN.
  • В некоторых местах могут глушить GPS — приложение может работать некорректно.
